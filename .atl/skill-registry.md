# Skill Registry - game-builder

## Project Standards
- **Component Patterns**: Logic in `use*.hook.ts`, JSX in components.
- **Styling**: Tailwind CSS only, no inline styles (except canvas dynamic values).
- **State**: Zustand with granular selectors.
- **Types**: Custom types in `src/types/`, no `any`.
- **Constants**: Upper snake case in `src/constants/`.

- `sdd-*`: Logic for SDD phases.
- `branch-pr`: PR workflow.
